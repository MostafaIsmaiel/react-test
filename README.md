# React Test Project

A modern React application built with Vite, TypeScript, and Tailwind CSS.

## Tech Stack

- **React 19.2** - Latest React with compiler support
- **TypeScript 5.9** - Type-safe development
- **Vite 7** - Fast build tool and dev server
- **Tailwind CSS 4** - Utility-first CSS framework
- **TanStack Router** - Type-safe routing
- **Radix UI** - Accessible component primitives
- **Prettier** - Code formatting with Tailwind plugin

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Linting

```bash
npm run lint
```

## Project Structure

```
src/
├── components/       # Reusable UI components
├── features/        # Feature-based modules
├── assets/          # Static assets
└── lib/            # Utility functions
```

## Features

- ✅ React Compiler enabled for automatic optimization
- ✅ Tailwind CSS v4 with Vite integration
- ✅ Component library with shadcn/ui
- ✅ Type-safe routing with TanStack Router
- ✅ ESLint + Prettier configuration
- ✅ Lucide React icons

## Configuration

### Prettier

Configured with Tailwind CSS plugin for automatic class sorting. See `.prettierrc` for settings.

### TypeScript

Strict type checking enabled. Separate configs for app and node environments.

## React Compiler

The React Compiler is enabled on this project for automatic performance optimizations. See [React Compiler documentation](https://react.dev/learn/react-compiler) for more information.
